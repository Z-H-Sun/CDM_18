ChemBioFinder Database Gateway HotLink shows which databases hosted at CambridgeSoft's ChemBioFinder.com contain data on a selected compound in an easy to use, clickable form which updates as you draw a chemical structure.

Information retrieved includes:
  • Databases and links to detailed records
  • Chemical names and synonyms
  • Chemical Identifiers (including InChI and SMILES Strings)

Requires an active Internet connection set to allow HTTPS traffic.
