Help for PerkinElmer Product Activation

 

Activate your software (Online)

Type your name, email address and activation code into the designated fields and click on the "Activate" button.  If the activation succeeds, you will receive a confirmation message. 

 

Problems with Activation 

If activation was unsuccessful, there are several possible reasons:

 

*  A firewall may be blocking access to the activation server.  If possible, temporarily disable your firewall and then try the activation again.

*  Your computer may need to install an SSL certificate.  You will get a special message with a solution to this problem directing you to our website, https://perkinelmer.flexnetoperations.com.  Simply visiting the site will provide you with the certificate.

*  You might not be connected to the internet.  If so, try again after reconnecting to the internet.

*  The activation server might be down or temporarily busy.  In this case, try again later.

 

If the online activation failure still persists, you can choose one of the alternative activation methods described below by clicking on the "More Options >>" button (for Windows) or expanding the down arrow next to "More Options" (for Mac).

 

Activate by uploading a file

*  Select the "Activate or deactivate by uploading a file" option and click on the "Activate" button.  You will be prompted to save a local .BIN file to some location you choose (e.g., your Desktop).  Then close the Activation Dialog.  Continue installing the software to completion, if you haven't done so already.

*  Navigate to the following web site: https://perkinelmer.flexnetoperations.com and log in. If you have never logged in before, create an account using your organization’s email address. Once logged in, select "Devices">"Upload Capability Request".

* Select the "Choose File" option, navigate to your .BIN file and then select "Open".  Then select "Send" to upload this .BIN file.  The "Response.BIN" file is downloaded from the site to your computer.  This file should be placed in the @@fnetsfolder folder on your hard drive.  Then relaunch the software to activate the product.

Activate later

If you choose not to activate now (or are unable to because of a firewall or lack of an Internet connection etc.), you have a number of free launches of the product available before activation is required.  Select the "Activate later" option and click on the "Activate" button to continue.

 

Deactivation of the software

Each activation obtains a license to use the software for your computer.  Licenses can be moved to other computers.  This can be achieved by deactivating the software on a currently licensed computer and activating it on another.  This operation is especially important if only a few license seats have been purchased.  You can choose to deactivate the product by selecting either "Activate or Deactivate online" option or "Activate or deactivate by uploading a file" option. Follow the steps mentioned above for activating the product and click "Deactivate" instead of "Activate".
