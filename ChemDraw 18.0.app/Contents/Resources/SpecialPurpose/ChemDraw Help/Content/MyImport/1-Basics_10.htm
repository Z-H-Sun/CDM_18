Getting Started

This topics covered in this chapter will help you become familiar with the user interface:
  • About ChemDraw Tutorials: Introduces you to the Tutorials chapter and provides a link to this chapter. These tutorials provide step-by-step instructions to help you work with ChemDraw tools.
  • ChemDraw User Interface: Describes the commonly-used toolbars within the user interface, and provides information about docking and floating toolbars for your use.
  • Documents: Provides basic information about creating documents in ChemDraw, along with saving documents, using styles, deleting keywords, about autosave, and default file formats.

Two other helpful topics to help you get started are:

  • Preferences and Settings: Provides information to help you set your preferences and customize ChemDraw.
  • Shortcuts and Hotkeys: Describes the shortcuts and hotkeys available with ChemDraw.
