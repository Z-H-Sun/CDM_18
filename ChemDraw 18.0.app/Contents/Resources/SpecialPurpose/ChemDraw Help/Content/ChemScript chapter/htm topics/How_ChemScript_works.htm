How ChemScript works

The most fundamental use for ChemScript is to read data from one source, modify the data using a script, and write the modified data to another location. You can retrieve data from or write to almost any database, file(s), or application.

 

A ChemScript script can retrieve data from one source, modify the data, and write it to another location.

How the data is modified is determined entirely by the script. The script can delete data, calculate or add new data, or edit existing data. The data can be either text, structures, or both.
Since ChemScript scripts are like any other Python scripts or .NET program, you can execute the scripts using either the Windows command line or any development environment.
